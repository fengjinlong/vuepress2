## linux
```
ls cd 
退回
cd / 根目录
cd ..
cd../..

创建aaa
mkdir aaa

cd aaa
mkdir bbb

复制文件
cp 1.txt aaa/1.txt
复制文件夹
cp -R aaa ddd
当前目录
pwd
删除
rm 1.txt
删词目录
rm -r bbb

新建文件
touch index.js
编辑文件
vi index.js
查看
cat index.js
```

## windows
```
dir 查看当前所有
cd

创建文件夹
md aaa

cd..

复制
copy a.txt aaa
删除
del a.txt
改名字
rename 1.txt 2.txt

```
